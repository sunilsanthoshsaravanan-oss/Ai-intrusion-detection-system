# Ai-intrusion-detection-system
AI-Based Intrusion Detection System (IDS)


Team Name: Dont Watch Me
Team Members: Sunil Santhosh S, G Ritheesh

Project Overview:

This project is an AI-based Intrusion Detection System that monitors network activity in real time and detects cyber attacks. It classifies data as SAFE or ATTACK and generates instant alerts.

Objective:
Detect cyber attacks in real time
Monitor system/network activity
Provide instant alerts

Working:

Data is generated or received
AI logic checks the data
Classifies as SAFE or ATTACK
Alerts are shown if attack is detected
Results displayed in dashboard

Technologies Used:
Python
Streamlit
Basic AI logic
Random module

Features:
Real-time monitoring
Attack detection
Instant alert system
Simple dashboard

Applications:
Cybersecurity systems
Banking security
Cloud protection
IoT devices
